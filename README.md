## Calculadora de divisas
Proyecto realizado con Next.js 14 y Tailwind CSS. Api de [saldo.com.ar](https://saldo.com.ar/)


## Getting Started
Para correr el proyecto instalar las dependecias con:
```
npm i
```
Luego ejecutarlo con:
```
npm run dev
```
